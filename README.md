    - 2️⃣ What is the spread operator (...)?
    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    - 4️⃣ What is an arrow function?
    - 5️⃣ What are template literals?

---

1.  var is global or function scope. it is possible to re-declare and update variable,
    let and const is block scope. let is possible to update but re-declare not allowed and
    const can not possible any thing update or re-declare.

2.  spread operator (...) is a simple way to expand elements from array, string and objects.

3.  map(), filter(), and forEach() are array methods.
    map() change array elements and return new array. original array does not change.
    filter() selects array elements based on a condition and return new element
    forEach() apply function to each element. it is return undefined.

4.  Arrow function is a shorter and cleaner syntax of the traditional function.
    It is written with the arrow operator =>.

5.  Template literals use backticks (``) to define strings.
    It is a powerful method that can write multiline strings, allows variables in strings, supports HTML templates, and supports expression interpolation.
